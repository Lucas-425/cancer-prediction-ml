# Diagnostic Prédictif par Machine Learning

## Description
Pipeline complet de classification supervisée appliqué au diagnostic médical.

## Résultats
- Précision : **96,49%**
- Modèle : Random Forest (Scikit-learn)
- Variables : 30+ features après Feature Engineering

## Stack technique
- Python (Pandas, NumPy, Scikit-learn)
- Feature Engineering, StandardScaler
- Évaluation : F1-score, AUC-ROC, matrice de confusion

## Structure du projet
- `notebook.ipynb` : pipeline complet
- `data/` : données utilisées
- `models/` : modèle entraîné
